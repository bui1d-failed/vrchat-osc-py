# **vrchat-osc-py**

**this repo is used to store my vrchat osc python scripts**
#

## **Enable OSC First !!! (and how)**
1. Press R in game
2. Click options
3. Click OSC
4. Turn 'Enabled' ON
5. Done and launch script
